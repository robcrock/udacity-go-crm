CRM Backend

The application handles the following 5 operations for customers in the 'database':

  • Getting a single customer with a GET request to localhost:3000/customers/{id}
  • Getting all customers with a GET request to localhost:3000/customers
  • Creating a customer with a POST request to localhost:3000/customers
  • Updating a customer with a PATCH request to localhost:3000/customers/{id}
  • Deleting a customer with a DELETE request to localhost:3000/customers/{id}
