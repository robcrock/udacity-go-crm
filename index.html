CRM Backend

API Usage Overview

Here is a list of the 5 functions of the API and their corresponding endpoints:

  • Getting a single customer - GET 
    localhost:3000/customers/{id}
  • Getting all customers - GET 
    localhost:3000/customers
  • Creating a customer - POST 
    localhost:3000/customers
  • Updating an existing customer - PATCH 
    localhost:3000/customers/{id}
  • Deleting an existing customer - DELETE 
    localhost:3000/customers/{id}
Note: You will need to include a customer stuct with your POST request.

An example of a properly formatted customer struct looks like this:

      {
        "ID": 3,
        "Name": "Coco",
        "Role": "pet",
        "Email": "r@g.com",
        "Phone": "(555) 555-5555",
        "Contacted": false
      }

For the best experience it's recommended to use Postman to make the request listed above.
